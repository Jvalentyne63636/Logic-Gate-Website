Logic Gates

AND Gate

This is an AND Gate. It requires both inputs (A and B) to be positive in order for the output (Q) to be positive. If only one of the inputs are positive, the output will be negative as a result of both not being positive.

Truth Table

OR Gate

This is an OR gate. It requires either one of the inputs, either A or B to be positive for the output, Q, to be positive. Both results A and B can be positive to make Q a positive.

Truth Table

NOR Gate

This is a NOR gate. It requires both the inputs (A and B) to be negative for it to work. It is an inverse of the OR gate. This means if either of the inputs are positive, the output (Q) is positive so the inverse means that if either of the inputs are positive, the output is negative.

Truth Table

EXOR Gate

This is an Exclusive OR gate, it requires either of the inputs to be positive like a standard OR gate however, it doesn't allow both inputs to be positive so the output, Q, will remain negative. It is exclusively only tolerant for only 1 positive result either (10)or (01).

Truth Table

EXNOR

This is an Exclusive Not OR Gate. This gate works like an inverse for an EXOR gate. It needs both the inputs to be positive or negative to make the output positive. If only either of the inputs are positive, the output is negative.

Truth Table

NAND Gate

This is a NOT AND gate. It works as an inverse for the AND gate. It requires either of the inputs to be positive (Not both!) or both the inputs to be negative for the output to be positive.

Truth Table

NOT Gate

The Not Gate returns the opposite of what is inputted. For example, an input of 1 returns an output of 0.
